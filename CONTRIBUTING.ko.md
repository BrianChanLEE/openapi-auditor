# openapi-auditor 기여 가이드

openapi-auditor에 관심을 가져주셔서 감사합니다! 여러분의 기여가 오픈소스 커뮤니트를 더 좋게 만듭니다.

## 행동 강령 (Code of Conduct)

긍정적이고 포용적인 환경을 유지하기 위해 [행동 강령](CODE_OF_CONDUCT.md)을 준수해 주세요.

## 기여 방법

- **버그 제보**: 명확한 설명과 재현 단계를 포함하여 이슈를 생성해 주세요.
- **기능 제안**: 플랫폼 개선을 위한 모든 아이디어를 환영합니다.
- **풀 리퀘스트 (PR)**:
  1. 저장소를 포크합니다.
  2. 기능별 브랜치를 생성합니다.
  3. 테스트 통과 여부를 확인합니다.
  4. 변경 내용에 대한 상세한 설명과 함께 PR을 제출합니다.

## 개발 환경 설정

1. `npm install`
2. `npm run dev` (로컬 샘플로 테스트)
3. `npm run build` (컴파일 확인)
