# 보안 정책 (Security Policy)

## 지원 버전

보안 업데이트는 최신 메이저 버전에 대해서만 제공됩니다.

| 버전 | 지원 여부          |
| ---- | ----------------- |
| 1.x  | :white_check_mark: |
| < 1.0| :x:               |

## 취약점 보고

보안 취약점을 발견한 경우, 공개적으로 이슈를 생성하지 마시고 `security@example.com` (예시)으로 이메일을 보내주세요. 48시간 이내에 답변을 드릴 수 있도록 노력하겠습니다.
