# 리포트 형식

자동 생성되는 리포트 제품군은 사람이 검토하는 리포트와 머신 기반의 CI 파이프라인 연동을 위한 데이터를 모두 제공합니다.

## REPORT.md (사람용 리포트)

스테이크홀더 및 개발자들과 즉시 공유 가능한 수준의 마크다운 리포트입니다.

### 구성
1. **종합 요약**: 성공률, 응답 시간, 우선순위별 이슈 통계.
2. **주요 진단 결과 (P0 ~ P1)**: 즉각적인 수정이 필요한 치명적 결함에 대한 상세 카드.
3. **아티팩트 링크**: 각 실패 사례에는 상세 디버깅을 위한 개별 JSON 파일 링크가 포함됩니다.
4. **전체 리스크 분석 표**: 모든 테스트 케이스에 대한 정렬 가능한 리스트.

---

## REPORT.json (데이터 리포트)

진단 결과의 구조화된 버전으로, 커스텀 CI 스크립트나 대시보드 통합에 적합합니다.

```json
{
  "summary": {
    "total": 42,
    "success": 38,
    "failure": 4,
    "successRate": "90.5",
    "p0": 1,
    "p1": 2,
    "timestamp": "2024-02-17T..."
  },
  "results": [ ... ]
}
```

---

## artifacts/*.json (상세 로그 아티팩트)

실패한 모든 테스트에 대해 `artifacts/` 폴더 내에 개별 JSON 파일이 생성됩니다.
구성 요소:
- **마스킹된 헤더**: Bearer 토큰 및 쿠키 등이 자동으로 가려집니다.
- **전체 페이로드**: 테스트에 사용된 실제 요청 데이터.
- **상세 응답**: 상태 코드, 바디, 특정 에러 메시지.
- **수정 가이드**: 메인 리포트와 동일한 해결 방법 안내 포함.
