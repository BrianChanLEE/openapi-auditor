# Contributing to openapi-auditor

First off, thank you for considering contributing to openapi-auditor! It's people like you who make the open-source community such an amazing place to learn, inspire, and create.

## Code of Conduct

Help us maintain a positive and inclusive environment by following our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

- **Reporting Bugs**: Open an issue with a clear description and reproduction steps.
- **Suggesting Enhancements**: We welcome ideas to improve the platform.
- **Pull Requests**:
  1. Fork the repo.
  2. Create a feature branch.
  3. Ensure tests pass.
  4. Submit a PR with a detailed description.

## Development Setup

1. `npm install`
2. `npm run dev` (to test with local samples)
3. `npm run build` (to check compilation)
